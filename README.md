# ACCdr
**ACCdr** is an app designed for coding.
> [!WARNING]
> This app will not work on **AC Device** unless you have the ACCdr 0.8 app. That means if you're in AC Device, you will get an older version of ACCdr that will not work on other devices like **Android** or **Windows**.

---

# What does it feature?
## File extensions
1. `.zip`, `.7z`: Archives
2. `.fldr`: Folder (but actually an archive)
3. `.accdr`, `.js`, `.yaml`, ...: Coding files
4. `.txt`: Text document
5. `.md`: Markdown file
> [!TIP]
> ACCdr supports Markdown files!
6. `.reg`: Registery file
> [!NOTE]
> Registery files will show a $$\textcolor{warning}{warning}$$ before you run them. (since it modifies the Windows computer)
7. `.bat`: Batch file
8. `.sh`/`.shell`: Shell file
9. `db.accdr` (a file): Discord bot .accdr file
10. `.device`: Device file
> [!TIP]
> A `.device` file is basically a device file that can be ran on AC Device using **ACCdr 0.8**. Note that it's also coded in X, though that language is low-level.
(...and more depending on the version)
